## 1. Branch 확인방법

```
git branch
```

## 2. Branch 사용

```
git checkout 브랜치이름
```

## 3. Branch 새로 만들기

```
git branch 새로운 브랜치이름
```

## 4. Branch 삭제

```
git branch -D 브랜치이름
```

## 5. Branch 원격에서 올리려면

```
git push origin 브랜치이름
```

## 6. Branch 원격에서 삭제하려면

```
git push origin -D 브랜치이름
```
